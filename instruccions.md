# Instruccions principals de Git

A continuació tens les ordres bàsiques de Git amb una breu descripció i un exemple pràctic de com s'utilitzen.

---

# 1. `git init`
**Descripció:**  
Inicialitza un repositori Git nou a la carpeta actual.

**Exemple:**
```bash
git init
```

# 2 `git status`
**Descripció:**  
Muestra el estado del repositorio: archivos modificados, nuevos o pendientes de commit.

**Exemple:**
```bash
git status
```

# 3 `git add `
**Descripció:**  
Añade archivos al índice (staging area) para prepararlos para el commit

**Exemple:**
```bash
git add archivo.txt
```

# 4 `git commit `
**Descripció:**  
Guarda los cambios añadidos al índice en el repositorio local con un mensaje descriptivo.

**Exemple:**
```bash
git commit -m "Descripción de los cambios"
```

# 5 `git push `
**Descripció:**  
Envía los commits locales al repositorio remoto (por ejemplo, GitHub).

**Exemple:**
```bash
git push origin main
```
