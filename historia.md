# Historia de Git

**Origen i creador:**  
Git va ser creat per **Linus Torvalds** l'any **2005**, el mateix creador del nucli de Linux.  

**Motiu de la seva creació:**  
Es va desenvolupar per gestionar el codi font del projecte **Linux**, després que l'equip deixés d'utilitzar el sistema de control de versions BitKeeper. Linus necessitava una eina que fos **ràpida, eficient i capaç de treballar amb molts desenvolupadors de manera distribuïda**.  

**Objectiu i ús actual:**  
Git és un sistema de control de versions **distribuït**, que permet:  
- Registrar i gestionar els canvis en fitxers i projectes.  
- Treballar de manera col·laborativa amb altres desenvolupadors.  
- Gestionar diferents branques de desenvolupament de manera segura i eficient.  

Avui dia, Git és l'eina més utilitzada per a la gestió de projectes de programari, especialment en combinació amb plataformes com **GitHub, GitLab o Bitbucket**.
